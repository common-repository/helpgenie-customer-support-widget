= Beautiful FAQ Widget For Your Store =
Enhance your customer experience with HelpGenie. HelpGenie.io makes useful information and links easily accessible to your visitors so you can boost website conversions. You can customise what links and videos to show based which page the visitor is at.

== Installation ==

= Automatic installation =

1. Go to Plugins > Add New > Upload and select the .zip file from your hard disk
2. Click the "Install now" button
3. Activate the plugin through the 'Plugins' menu in WordPress

= Manual installation =

1. Upload the plugin folder to the `/wp-content/plugins/` directory via ftp
2. Activate the plugin through the 'Plugins' menu in WordPress
